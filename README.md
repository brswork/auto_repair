# AutoMotoLab — Сайт сервиса ремонта авто и мото

Минималистичный одностраничный сайт для сервиса ремонта автомобилей и мотоциклов. Дизайн современный, технологичный, с адаптивной версткой и простой интерактивностью.

Файловая структура:

- `index.html` — главная страница
- `css/styles.css` — стили
- `js/app.js` — простая логика: мобильное меню, плавный скролл, форма
- `assets/hero.svg`, `assets/favicon.svg` — иллюстрации

Как запустить локально

1) Откройте в браузере напрямую:

```bash
xdg-open index.html
```

2) Или запустите локальный HTTP-сервер (рекомендуется для корректной работы некоторых браузеров):

```bash
# Python 3
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Кастомизация

- Подмените текст и контактные данные в `index.html`.
- Замените `assets/hero.svg` на собственное изображение.
- Для интеграции с бэкендом отправку формы нужно заменить вызов `alert(...)` в `js/app.js` на `fetch()` к API.

Новые блоки

- `partners` — логотипы и названия партнёров (placeholder блок для брендов).
- `gallery` — простая галерея работ (замените на реальные фотографии в `assets/`).
- `faq` — список часто задаваемых вопросов с аккордеоном (реализовано в `js/app.js`).
- `contacts` — контакты, режим работы и встроенная карта (iframe). Адрес/координаты можно заменить в `index.html`.
